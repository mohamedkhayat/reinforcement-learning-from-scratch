# Reinforcement Learning From Scratch

Implementations of Reinforcement Learning algorithms built from first principles (Python/NumPy) and later migrating to Deep RL (JAX/PyTorch).

**Goal:** Implement agents without abstraction libraries (e.g., Stable-Baselines) to understand the underlying math.

## References
*   *Reinforcement Learning: An Introduction* (Sutton & Barto) - For tabular methods.
*   *Deep Reinforcement Learning Hands-On* - For Deep RL architectures.
